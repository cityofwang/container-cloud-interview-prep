# Go 过线编码

目标：合面/编码轮能写出可编译的小并发题，不因「完全不会写 Go」挂掉。不做算法岗刷题。

## 范围

- 并发小题：context 取消、WaitGroup、Mutex/atomic、channel/select
- 字符串 / 哈希、简单限流或 worker pool（后续练习再加）
- 每周 1～2 道；对应能力维度 **X1**

## 与 Shell C-01 的关系

- 本目录以 **Go 为主**计数；进三面前至少完成 **≥3 次**（可重复同类题改写）
- 原 `05-coding-line/` 的 Shell **C-01** 最多计 **1 次**，其余次数须为 Go（本目录 G-xx）
