# 错题本

路径约定见 `docs/superpowers/specs/2026-07-22-spaced-review-and-china-sources-design.md` §2.1。  
口令：`看错题本` / `错题回炉`。

## 入库规则（摘要）

- 自评「不会」、模拟 ≤2 分、或存在硬伤 → **必入**
- 自评「半会」或模拟 =3 → **建议入**（默认入，学员可标跳过）
- 仅表达略生但主线对（4 分）→ 不入，除非学员要求
- 巩固出库：同一题连续 2 次复现得分 ≥4 且无硬伤 → 标「已巩固」
