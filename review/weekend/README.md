# 周末训练包

设计见：`docs/superpowers/specs/2026-07-22-weekend-sprint-scheduling-design.md`

## 时间盒

- **平日**：默认只错题（`看错题本` / `错题回炉`），≤30 分钟
- **周末**：约 3～4 小时；发口令 `周末开练` 生成当次包
- **碎片**：`我有空 30分钟` / `我有空 1小时`

## 包类型（教练按能力档选择一种）

| 类型 | 何时 | 重点 |
|------|------|------|
| D 摸底/诊断 | Go 摸底未完成 | 摸底 + 轻量容器补洞 |
| B 补短板 | L1～L2 | 回炉 + 推荐小节 |
| M 合面短模 | 冲 L3 | 合面 + 复盘 + 故事 |
| S 冲刺 | 距面试 ≤14 天 | 高频 + 错题 + 故事，不开新冷门 |

## 当次计划文件

每次 `周末开练` 写入：

`YYYY-MM-DD-plan.md`

建议字段：日期、包类型、目标档、时间盒、题单、错题复现 id、结束后更新项（RESULT/评估/错题本）。
