## R2-P01 CrashLoop / OOM 路径加深

**题目：** CrashLoop 与 OOMKilled 排查路径有何不同？如何用对比说明 CPU throttle 与 Memory OOM？

**要点：** 日志/退出码 vs cgroup OOM 事件；throttle 表现为延迟而非必杀；limits 与观测。  
**对比点：** 应用挂 vs 被内核杀；CPU 限 vs 内存限  
**挂钩：** S2、S5

## R2-P02 混部干扰与隔离选型

**题目：** 在离线混部时「干扰」如何观测？隔离手段如何选型（cgroup / 污点池 / 反亲和 / 配额）？

**要点：** 延迟/抖动/窃取指标；先观测再加硬隔离；与业务解释 blast radius。  
**对比点：** 软限制 vs 硬隔离；池化 vs 打散  
**挂钩：** S1（主）

## R2-P03 CSI 部署与常见坑

**题目：** CSI 相关组件常见部署形态？挂载失败你先看什么？

**要点：** controller + node plugin（常 DaemonSet）；RBAC、挂载超时、拓扑、权限；灰度发布插件。  
**对比点：** 控制面插件故障 vs 节点插件故障  
**挂钩：** S4

## R2-P04 NotReady 节点

**题目：** 节点 NotReady，分层怎么查？何时驱逐、何时先保业务？

**要点：** kubelet/运行时/磁盘/网络；驱逐策略与业务影响；止血 vs 根治。  
**对比点：** 节点问题 vs 控制面误判  
**挂钩：** S5
