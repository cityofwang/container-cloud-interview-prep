# 周末攻坚 + 平日错题 + 能力动态调速

- 日期：2026-07-22
- 状态：已确认；骨架已落地（README / review/weekend）
- 适用范围：全仓训练节奏（容器 + Go + 合面 + 错题 + 雷达）
- 关联：评估体系 `2026-07-22-assessment-and-unlock-gates-design.md`；错题/雷达 `2026-07-22-spaced-review-and-china-sources-design.md`；Go 双轨 `2026-07-22-golang-dual-track-design.md`

## 1. 目标与时间约束

- **目标**：尽快达到「可通过面试」水准（优先 **L3**，再冲 **L4**），而不是刷完题库。
- **时间**：
  - **平日**：默认只有错题（`看错题本` / `错题回炉`），单次建议 ≤30 分钟。
  - **周末**：约 **半天 3～4 小时**（方案 A）；是否开练由学员发口令（方案 D 弹性）。
- **节奏方案**：**方案 1 — 能力驱动周末包**。

## 2. 口令

| 口令 | 作用 |
|------|------|
| `周末开练` | 按当前能力档生成并开始本周末 3～4h 训练包 |
| `今天只能看错题` | 强制进入平日模式（即使周末） |
| `我有空 30分钟` / `我有空 1小时` | 弹性碎片时间：只回炉或极短补洞，不加新大课 |
| `能力评估` | 看当前档与本周该攻什么 |
| 既有口令 | `开始 Go 摸底` / `开始一面题库` / `错题回炉` / `扫岗位` 等仍有效 |

## 3. 能力动态调速（感知规则）

每次开练前读取：`ASSESSMENT.md` 当前档、错题未巩固数、最近一次模拟/摸底正确率。

| 信号 | 调速动作 |
|------|----------|
| 档 ≤ L1，或正确率 &lt; 50% | 周末包 **70% 回炉 + 基础**，少开新考点；暂缓合面加长 |
| L2，正确率 50–70% | 周末包 **50% 新课薄弱节 + 50% 回炉/短模** |
| L3，正确率 ≥70%，未巩固错题 ≤5 | 周末包以 **合面短模 + 专长深挖（混部或并发）** 为主 |
| 正确率掉档（连续 2 次 &lt;60%） | 下一包强制「回炉周」，进阶继续锁 |
| 距面试 ≤14 天（学员告知） | 切换「冲刺包」：只高频 + 错题 + 故事口述，停开新冷门节 |

**尽快过面的内容优先级（默认顺序）**

1. Go 摸底 → 按 RESULT 开 basics 或 concurrency（先补短板）  
2. 容器一面过线（已有题包）  
3. 第一次 **合面短模**（容器+Go，控在 60–75 分钟）  
4. 二面主战场：排障 + 混部/cgroup 故事 + Go 并发  
5. 达 L3 后再谈 E1–E4；达 L4 再谈 E5 谈薪  

不做：平日塞新大课；周末一次塞两种全新大节。

## 4. 周末 3～4h 包模板

由教练按调速选 **一种主型**（勿叠满）：

**类型 D（摸底/诊断）** — 尚未完成 Go 摸底时优先  
- 0:15 热身 + 看错题本  
- 2:00 Go 摸底闭卷对练 + 填 RESULT / 错题入库 / 更新评估  
- 0:45 容器一面薄弱 3～5 题开卷补洞  
- 0:15 下周平日回炉清单  

**类型 B（补短板）** — L1～L2  
- 0:20 错题回炉（3/7 天到期优先）  
- 2:00 推荐小节第一批（如 concurrency）开卷+短模  
- 0:40 容器一面补洞或 G-01 编码一题  
- 0:15 更新评估与下周清单  

**类型 M（合面短模）** — 冲 L3  
- 0:15 错题必带抽题预习  
- 1:15 合面模拟（约 6 容器 + 4 Go，含 2～4 错题复现）  
- 0:45 复盘改口述  
- 0:30 专长故事卡补全（S1/S2/S3 选一）  
- 0:15 清单  

**类型 S（冲刺）** — 距面试 ≤14 天  
- 错题 + 高频合面变形 + 故事口述；禁止新冷门专题  

每次 `周末开练` 输出一张「本包卡片」写入：  
`review/weekend/YYYY-MM-DD-plan.md`（日期、类型、目标档、题单、时间盒）。

## 5. 平日模式

- 默认：只维护错题本；到期 3/7 天则提示 `错题回炉`  
- 单次 ≤30 分钟；不做合面、不开新小节题包  
- 学员若工作日发 `周末开练`：先确认是否误触，建议改为 `我有空 30分钟`

## 6. 与既有体系的关系

- 评估 L1–L5、进阶解锁、雷达校准、国内题源、错题混合复现 —— **全部继承**  
- 本 spec 只约束 **何时练什么、练多久**  
- 岗位雷达：建议挂在周末包开头 10 分钟或另约；不占平日错题额度
