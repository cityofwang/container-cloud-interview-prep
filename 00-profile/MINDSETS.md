# 横切思维：对比 / 选型 + 角色靶心

学员确认：2026-07-23。  
**不进八维加权**；贯穿开题、笔记、排考点、合面追问与故事深挖。  
口令：`看思维清单` / `对比追问` / `考察说明`。操盘时助手默认带「对比点」；模拟时按 §3 考察。

---

## 1. 对比思维（主轴）

| 类型 | 用法 | 例子 |
|------|------|------|
| 概念对比 | 易混点成对讲，禁止只讲单边 | `%w` vs `%v`；有/无缓冲 channel；requests vs limits |
| 场景对比 | 何时 A / 何时 B / 误用代价 | Mutex vs channel；扩容 vs 限流；重启 vs 熔断 |
| 竞品/选型对比 | 方案取舍、边界、回滚 | CSI 灰度；混部隔离手段；监控自建 vs 托管 |
| 前后对比 | 故事用改前/改后 | 事件风暴前 vs 限流后；泄漏前 vs pprof 后 |
| 层级对比 | 现象 → 机制 → 边界，不混层 | Service 不通：先连通性再 CNI 原理 |

**落地检查：** 新题/新笔记至少有一行「对比点」；合面追问优先打对比。

## 2. 培训官思维（偏「教」）

| 思维 | 含义 |
|------|------|
| 目标对齐 | 服务当前画像（现 L3），不炫技超纲 |
| 诊断先于灌输 | 先摸底/错题，再讲；不会才写详解 |
| 对比与追问 | 不满足背定义；追「和 X 比呢」「线上怎么证」 |
| 国内题感 | 岗+题双雷达；不编造「某厂原题」 |
| 时间盒诚实 | 平日不塞大课；人话映射口令 |
| 证据导向 | 量尺打分、错题入库、改期有理由 |
| 不编造经历 | 只深挖真实做过的 |
| 选型官口吻 | 「如果让你拍板」练判断 ≠ 上完整架构课 |

## 3. 面试官考察思维（偏「面」）

学员确认 2026-07-23。助手在摸底对练、闭卷模拟、合面中**必须按此考察**。

| 思维 | 考察时怎么用 |
|------|----------------|
| **证据优先** | 听机制、例子、验证手段；不听空形容词 |
| **分层探测** | 主线（会不会）→ 对比/边界 → 线上怎么证；一层过了再加压 |
| **对比施压** | 默认追「和 X 比呢」「为何不选 B」——验真懂还是背过 |
| **生产可证伪** | 「怎么证明」「看什么指标/事件」；说不清则降档 |
| **硬伤敏感** | 概念反了且不自知 → 低分+入库；主线对仅表达生 → 不重罚 |
| **岗位对齐** | 题感贴容器云+Go 与当前 L 档；不拿 L5 谈薪题为难 L3 |
| **经历守恒** | 只深挖真实做过的；不会则要「边界+排查路径」 |
| **时间与信号** | 短时看结构与主线；超时看取舍是否清醒，不唯代码完美 |
| **公平量尺** | 1–5 对照仓库标准；不因投机抬分 |
| **可迁移判断** | 换问法仍稳 = 会；只会原题表述 = 半会 |

**出题 / 排考点额外三条**

1. 诊断先于刁难（先暴露缺口，再针对性加压）  
2. 一卷至少含对比或选型问法  
3. 错题复现优先于堆新冷门  

**和 §2 的分工：** §2 管怎么教与组课；§3 管怎么面、怎么打分与是否过线。

## 4. 优秀面试者 / 工作者靶心

### 思维（软骨架）

- 对比与选型：取舍、代价、回滚说得清  
- 分层排查：影响面 → 假设 → 验证 → 止血  
- 边界诚实：不懂给边界 + 学习/排查路径  
- 可验证：指标、事件、pprof、变更记录  
- 系统反馈：复盘 → 防再发（告警/发布/容量）  
- 沟通影响面：对业务/值班说清 blast radius（观察维 O3）

### 能力（硬技能，对本岗）

- 容器/K8s 生产运维 + 混部/cgroup/CSI 深挖  
- Go 基础与并发过线 + 云原生直觉  
- 编码过线（非算法岗）  
- 结构化表达（四段式 / 排障 / STAR）

与八维、O1–O7、`STAGE-PORTRAITS.md` 一致；本文件是**横切要求**。
