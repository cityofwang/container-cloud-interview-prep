# 对话口令表（权威清单）

> **维护规则：** 以后每新增/废弃一口令，必须同步改本表，并更新 `README.md` 口令行；若影响续聊，再改 `CONTINUITY.md`。  
> 关联设计：`docs/superpowers/specs/2026-07-23-stage-portraits-and-question-sources-design.md`

最后更新：2026-07-23

## 时间盒 / 节奏

| 口令 | 作用 | 典型时机 |
|------|------|----------|
| `周末开练` | 按能力档组 3～4h 周末包 | 周末主战场 |
| `今天只能看错题` | 强制平日模式（即使周末） | 偶发 |
| `我有空 30分钟` | 碎片：回炉或极短补洞 | 平日常用 |
| `我有空 1小时` | 稍长碎片（仍非新大课） | 平日 |

## Go / 容器训练

| 口令 | 作用 | 典型时机 |
|------|------|----------|
| `开始 Go 摸底` | 开或续 Go 摸底对练 | **当前 P0 首选（续 GO-D-07）** |
| `开始一面题库` | 容器一面开卷 | P0 后 / 并行 |
| `开始模拟` | 闭卷模拟（单面或合面视进度） | 开卷比例够后 |
| `复盘` | 模拟后复盘 | 模拟后 |
| `补洞` | 针对薄弱点补口述/笔记 | 复盘后 |
| `编码练一题` | Go/过线编码一题 | P1～P2 |
| `练场景题` / `练一道场景题` | `05-scenario-line` 容量/限流/一亿邮件类 | 穿插；宁多勿漏 |
| `过关检查` | 对照该面过关线检查 | 模拟后 |

## 错题 / 笔记 / 图谱

| 口令 | 作用 | 典型时机 |
|------|------|----------|
| `错题回炉` | 按 3/7 天规则抽错题小测 | 平日首选 |
| `看错题本` | 只列出未巩固条目 | 查看 |
| `看知识图谱` | 打开知识点状态表 | 随时 |
| `看思维清单` | 打开对比/选型、培训官、**考察与选人标准**、学员靶心 | 随时 |
| `看流程清单` | K8s 关键流程熟练度（Pod 创建等） | 随时 |
| `对比追问` | 本场追问优先打对比/选型 | 对练/合面中 |
| `考察说明` | 口述考察十条 + 选人标准 | 随时 |
| `看 L3 排期` | 打开 P0–P3 与达标窗 | 随时 |
| `补笔记 <主题>` | 为不会的点写/补详解笔记 | 按需 |
| `看阶段画像` | 打开 L1–L5/Offer 能力画像 | 随时 |

## 评估 / 雷达

| 口令 | 作用 | 典型时机 |
|------|------|----------|
| `能力评估` | 输出当前档、八维、距下一档 | 随时 |
| `扫岗位` | **仅 JD** 招聘雷达 → `review/job-market/` | 约每周或按需 |
| `看岗位雷达` | 打开最近一次 JD 报告 | 扫岗后 |
| `扫题源` | **牛客/GitHub·笔记/面经** → `review/question-sources/` | 约每周或按需 |
| `看题源雷达` | 打开最近一次题源报告 | 扫题源后 |
| `按雷达校准评估` | 确认后改评估权重/门槛 | 报告有建议时 |
| `按报告补题` / `按题源报告补题` | 确认后把报告建议写入题包 | 扫岗/扫题源后 |

## 纠偏 / 续聊 / 操盘

| 口令 | 作用 |
|------|------|
| `请严格按 CONTINUITY.md 与 docs/superpowers/specs/ 下已确认设计执行。` | 助手跑偏时纠偏 |
| `按操盘规则继续` | 先给下一步建议（及等价口令）再开练；学员不必背口令 |
| （人话）有空半小时 / 周末了 / 好累只看下次 / 按 CONTINUITY 继续 | 由助手映射到上表动作，见 `CONTINUITY.md`「操盘规则」 |

> **操盘约定：** 口令给助手用；学员用人话即可。完整规则在 `CONTINUITY.md`。
