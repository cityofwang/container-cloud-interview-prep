# 阶段能力画像（目标对象）

量尺数字以 `ASSESSMENT.md` 与 assessment spec 为准；本文件写清**每一档「长什么样」**。  
当前训练主目标：**L3**。口令：`看阶段画像`。

最后更新：2026-07-23

---

## L1｜打地基

**一句话：** 容器基础对象能背，但闭卷易糊；Go 尚无体系；还不宜稳定投对口技术一面。

| 维度 | 画像 |
|------|------|
| 分数感 | 综合加权 &lt; 3.0 或正确率 &lt; 50% |
| 容器 | Pod/Service 名词知道，排障说不成路径 |
| Go | 能改脚本；error/并发讲不清 |
| 故事 | 经历有，但追问一层就空 |
| 编码 | 现场题易慌，无超时/取消意识 |
| 对标感 | 内推简历关都悬；需要先摸底+basics+一面开卷 |

**本阶段该做什么：** Go 摸底收口、错题回炉、一面开卷；不谈薪、不合面加长。

---

## L2｜一面可面

**一句话：** 对口初筛/一面有机会过；合面与深挖仍不稳。

| 维度 | 画像 |
|------|------|
| 分数感 | 加权 ≥ 3.0，正确率 ≥ 60%，无高频硬伤 |
| 容器 C1 | 一面对象与探针/QoS/滚动能四段式说完 |
| Go G1 | basics 主线过线；仍可能被 interface/error 链打穿 |
| 故事 | 能 STAR，深度不够 |
| 编码 X1 | 能写小函数，并发题勉强 |
| 对标感 | 可投托管 K8s/容器运维初面；大厂混部深挖易露怯 |

**出口：** 一面开卷达标 + 可约短模；错题未巩固开始下降。

---

## L3｜合面稳定（当前主目标）

**一句话：** 中高级技术轮站得住——容器+Go 合面短模无明显硬伤，专长或并发至少一头能深挖。

| 维度 | 画像 |
|------|------|
| 分数感 | 加权 ≥ 3.5，正确率 ≥ 70%，未巩固错题 ≤ 5；**C3 或 G2 ≥ 3.5** |
| 容器 C1–C2 | 排障有路径；监控/事件能挂钩生产 |
| 专长 C3 | 混部/cgroup/CSI **可被追问 2～3 层**（观察维 O6） |
| Go G1–G2 | error 链、泄漏、channel/context 主线过关；合面 Go 题均分够线 |
| G3 | 事件/List-Watch 直觉即可，不强制 Operator 大作 |
| 编码 X1 | 超时取消类小题能口述+落主路径；进三面前练习次数达标趋势 |
| 表达 X2 | 四段式/排障模板稳定；合面 60–75min 后半不崩（O7） |
| 对标感 | 云厂商/大厂容器云·平台向技术轮可冲；谈薪与高定价仍早 |

**尚未要求：** 谈薪策略、完整作品集包装、Mesh/多集群（除非雷达连续高频）。

**预计窗口：** 见 `review/roadmap/L3-plan.md`（动态）。

## L4｜可冲更高技术定价

**一句话：** 技术定价上移的主依据——专长深 + Go/编码稳，正确率高。

| 维度 | 画像 |
|------|------|
| 分数感 | 加权 ≥ 4.0，正确率 ≥ 80%；C3≥4.0 且 (G2≥3.5 或 G3≥3.5)；X1≥3.5 |
| 容器/专长 | 混部或 CSI/资源方案能讲边界与取舍；变更灰度回滚口述清晰（O2） |
| Go | 并发排障直觉强；云原生 Go（client-go/informer）能设计小题 |
| 编码 | 合面现场题不挂，边界能补 |
| 对标感 | 冲「高级/资深」技术轮与更高薪 JD 的技术侧条件 |

**细排期：** L3 达标后再开；本阶段不提前堆谈薪。

---

## L5｜谈薪准备就绪

**一句话：** L4 + 进阶模块（简历/JD 对齐、故事商品化、谈薪包等 E*）完成。

| 维度 | 画像 |
|------|------|
| 分数感 | 达 L4 + 约定进阶完成 |
| 软性 | 动机、稳定性、协作值班表达过关 |
| 对标感 | 可加速投递与谈薪；技术侧不再是主阻塞 |

**锁定：** E5 谈薪默认 L4 前不解锁（策略 D）。

---

## Offer 指向（非单独 L 档）

**一句话：** 在 L3/L4 技术达标基础上，完成投递节奏、面试复盘闭环与谈薪；结果受市场与运气影响，仓库只保证「人到位」。

| 要素 | 画像 |
|------|------|
| 技术 | 至少 L3；冲高薪岗建议 L4 |
| 市场 | 岗位雷达 + 题源雷达持续校准 |
| 过程 | 每次面试复盘入库错题；不因单次挂而推翻 L 档 |
| 不做 | 把 Offer 日期写成硬承诺；不因 Offer 焦虑降低 L3 门槛 |
