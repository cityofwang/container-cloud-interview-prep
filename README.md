## 怎么用

对话口令：`开始一面题库` / `开始模拟` / `复盘` / `补洞` / `编码练一题` / `过关检查` / `开始 Go 摸底` / `错题回炉` / `看错题本` / `扫岗位` / `看岗位雷达` / `能力评估` / `按雷达校准评估`

每面四步：开卷题库 → 闭卷模拟 → 复盘补洞 → 过关检查

作答写法见 `00-profile/ANSWERING.md`；能力雷达见 `00-profile/ASSESSMENT.md`。

## 目录

- `00-profile/` 背景、故事卡、作答写法、能力评估
- `01-round1/` … `04-round4/` 各面题库
- `05-coding-line/` 过线编码（Shell 等）
- `06-golang/` Go 双轨（摸底/小节/编码）
- `review/` 错题本与岗位雷达
- `mocks/` 模拟记录
- `docs/superpowers/` 设计与计划
